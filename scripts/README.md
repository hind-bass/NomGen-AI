# Scripts utilitaires

Exécuter depuis la **racine du projet** :

| Script | Commande | Rôle |
|--------|----------|------|
| `split_dataset.py` | `python scripts/split_dataset.py` | Divise `data/dataset_*.txt` en fichiers par secteur/type |
| `check_dataset.py` | `python scripts/check_dataset.py` | Contrôle qualité des fichiers `data/` |
| `generate_names.py` | `python scripts/generate_names.py` | Démo CLI Mode A + Mode B |
| `reset_db.py` | `python scripts/reset_db.py` | Réinitialise SQLite (`backend/nomgen.db`) |
| `filter.py` | `python scripts/filter.py` | Expérience de filtrage de noms (notebooks) |
